# csc302fa24-DeMinico-project

My project is going to be a sports app that users can add their favorite players to and see their stats. 

Currently their an API which will be used to get the info for the players for each user.

The users info is stored using db.php which makes a database to store their info on 

Their is a sign in screen and a home screen which are both very basic at the moment
The home screen will have info for the users and their favorite/tracked players and/or the most recent events.
