# Sports App

## Overview
My project is a sports app that allows users to add their favorite players and view their statistics. The app connects to an API to retrieve player information for each user.

## Features
- **User Authentication**: Users can sign in and manage their profiles.
- **Player Tracking**: Users can add their favorite players and track their stats.
- **Home Screen**: Displays user information and updates on favorite or tracked players, as well as recent events.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: PHP
- **Database**: SQLite (managed by db.php)
- **APIs**: Utilizes an external API to fetch player information.
